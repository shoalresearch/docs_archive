---
title: 'Get Signals by Category'
api: 'GET /v1/signal/byCategory'
description: 'Get signal items filtered by event category'
---

## Query Parameters
- `category` (string, required) — Event category (e.g. `partnership`, `regulation_legal`, `product_development`, `market_update`, `security_incident`)

### Request
```bash cURL
curl -X GET "https://api.shoal.xyz/v1/signal/byCategory?category=partnership" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### Response
Same as `/v1/signal/all` but filtered by category.

### Errors
- 400 if `category` missing
- 500 on internal server error
