---
title: 'Get All Radar'
api: 'GET /v1/radar/all'
description: 'Get all radar events'
---

## Query Parameters
- `limit` (integer, default: 50)
- `offset` (integer, default: 0)

### Request
```bash cURL
curl -X GET "https://api.shoal.xyz/v1/radar/all?limit=50&offset=0" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
