---
title: 'Get Events by Organization Name'
api: 'GET /v1/events/byOrganizationName'
description: 'Get events for a specific organization by name or alias'
---

## Query Parameters
- `name` (string, required) — Organization name or alias

### Request
```bash cURL
curl -X GET "https://api.shoal.xyz/v1/events/byOrganizationName?name=Org" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### Response
Same as `/v1/events/byOrganizationId`.

### Errors
- 400 if `name` missing
- 500 on internal server error
