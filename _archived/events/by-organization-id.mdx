---
title: 'Get Events by Organization ID'
api: 'GET /v1/events/byOrganizationId'
description: 'Get events for a specific organization by ID'
---

## Query Parameters
- `id` (integer, required) — Organization ID

### Request
```bash cURL
curl -X GET "https://api.shoal.xyz/v1/events/byOrganizationId?id=1" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### Response
Same as `/v1/events/all` but filtered for one organization.

### Errors
- 400 if `id` missing
- 500 on internal server error
