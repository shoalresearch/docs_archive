---
title: 'Get All Markets'
api: 'GET /v1/polymarket/all'
description: 'Get all markets with events'
---

## Query Parameters
- `limit` (integer, default: 50) — Number of organizations to return
- `offset` (integer, default: 0) — Pagination offset

### Request
```bash cURL
curl -X GET "https://api.shoal.xyz/v1/polymarket/all?limit=50&offset=0" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
