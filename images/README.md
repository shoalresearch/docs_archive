# Docs Images

Place your brand assets here and reference them from `docs.json` using paths like `/images/...`.

Recommended files:
- `/images/logo-light.svg` – Logo for light theme (SVG preferred)
- `/images/logo-dark.svg` – Logo for dark theme (SVG preferred)
- `/images/favicon.svg` – Favicon (SVG or 32x32 PNG)

Notes:
- Keep logo height small (24–32px) for navbar readability.
- Use high-contrast variants for light/dark themes.
- If you only have one logo, point both `logo.light` and `logo.dark` to the same file.
